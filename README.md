# Projet finance ESG
## Prérequis

- Python 3.12+

## Installation

### 1. Installer uv

```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

Puis relancer le terminal ou exécuter :

```bash
source ~/.bashrc   # Linux
source ~/.zshrc    # macOS
```

### 2. Installer les dépendances

```bash
uv sync
```

### 3. Lancer le projet

```bash
uv run main.py
```
